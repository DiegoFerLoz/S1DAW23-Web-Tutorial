ENLACES

Los enlaces a otras páginas se crean con el comando "a".

Para mayor comodidad se introduce dentro de un párrafo, esto da mayor control a la hora de modificar el texto del enlace.

Tipos de Enlaces

Un enlace a una página que tenemos en la carpeta

En este caso solo tenemos que pasar el nombre del fichero html que queremos ejecutar.

Este enlace va a Home.

Un enlace a una página de internet, en este caso EducamosCLM

Para enlazar una página web cualquiera, solo necesitamos su dirección.

Este enlace va a EducamosCLM.

Un enlace que se abre en una pestaña nueva

Si añadimos "target="_blank"" al enlace, este aparecerá en una nueva pestaña.

Este enlace se abre en otra pestaña.

Un enlace que ejecuta un programa de javascript (en este caso una simple orden de alert)

Se pueden ejecutar programas en javascript, en este caso se ejecuta un alert.

Este enlace ejecuta un programa javascript.

Un enlace que manda un email (lo intenta)

También se puede mandar un correo a un destinatario. Sin embargo, solo se ejecuta la órden de comenzar a escribir el correo, el cuerpo hay que definirlo manualmente.

Este enlace trata de mandar un correo.

Un enlace representado por una imagen

En lugar de texto, se puede utilizar una imagen. También se ha centrado la imagen.
