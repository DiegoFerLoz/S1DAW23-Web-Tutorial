MULTIMEDIA

Con las etiquetas "video" y "audio" se puede añadir contenido multimedia a una página web.

Utilizando atributos como "controls" o "autoplay" se puede customizar cómo se visualiza el vídeo.

Además, como existen infinidad de exploradores distintos y no todos reproducen los mismos tipos de archivo, es buena idea añadir múltiples tipos de archivo para que cada explorador reproduzca el tenga por defecto.
