LISTAS

Las listas se crean con los comandos "ul" (para listas desordenadas), "ol" (listas ordenadas), o "dl (para listas de definición).

Tipos de listas

Listas desordenadas

Las listas desordenadas (sin orden concreto) se crean con el comando "ul". Voy a crear una lista con 3 elementos:

  • Patata
  • Croqueta
  • Azuuul

Por defecto, los elementos se tabulan con círculos.

Listas ordenadas

Las listas ordenadas se crean con el comando "ol". Vamos a crear la misma lista, pero ordenada:

  1. Patata
  2. Croqueta
  3. Azuuul

En este caso, por defecto, los elementos se tabulan y se añaden números que ordenan la lista.

Listas de varios niveles

En cada elemento de una lista, ya sea ordenada o desordenada, se puede introducir otra lista, para así crear listas de múltiples niveles.
Vamos a crear una lista desordenada con 3 niveles:

  • Patata
    • Frita
    • Cocida
  • Croqueta
  • Azuuul
    • Claro
      • Celeste
      • Cian
      • Algún otro tono de azul claro
    • Oscuro
      • Marino
      • Lapislázuli
      • No se mucho de colores

Por defecto se asignan círculos al primer nivel, discos al segundo nivel, y cuadrados al tercer nivel. Esto se puede cambiar a gusto del programador, es lo que vamos a hacer a continuación.

Las listas multinivel pueden ser ordenadas o desordenadas, funcionan igual. Si se mezclan, los símbolos variarán.

Lista customizada

De nuevo, se puede customizar tanto las listas ordenadas como las desordenadas. Para ello, se establecen normas en el .css que afectarán a uno u otro nivel.

En este caso vamos a customizar la lista multinivel desordenada del ejemplo anterior:

  • Patata
    • Frita
    • Cocida
  • Croqueta
  • Azuuul
    • Claro
      • Celeste
      • Cian
      • Algún otro tono de azul claro
    • Oscuro
      • Marino
      • Lapislázuli
      • No se mucho de colores

En este caso le he puesto un cuadrado al primer nivel, números romanos al segundo nivel, y números decimales al tercer nivel.
Para ello se introduce la lista en una caja ("div"), donde se le asigna un id, que será único para esa caja y lista. Después, en el .css se añaden las reglas correspondientes a cada nivel.

Lista customizada con imágen externa

Además de poder cambiar el elemento que se asigna a cada nivel, se puede establecer una imágen externa que se asigne a cierto nivel de la lista.

Utilizando el archivo css, se crea otra clase que contiene "list-style-image: url(ruta)", con la ruta de la imágen, y se asigna de la misma manera.

  • Patata
    • Frita
    • Cocida
  • Croqueta
  • Azuuul
    • Claro
      • Celeste
      • Cian
      • Algún otro tono de azul claro
    • Oscuro
      • Marino
      • Lapislázuli
      • No se mucho de colores

Lista de definición

Las listas de definición se crean con el comando "dl", y sirven para establecer una lista de términos con sus definiciones.
Vamos a crear una lista de definición con tres términos:

Patata
Se come
Croqueta
Se come
Azul
Se trata de un color frío. Existen muchas variantes de este color, como el cian, celeste, etc.
