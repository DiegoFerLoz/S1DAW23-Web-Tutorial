Tipos de tablas

Tabla de encabezado de fila

Las tablas se crean con el comando "table". Se suelen añadir el número de filas y columnas después del comando.
Por ejemplo, al escribir el comando "table>(tr>th*5)+(tr>td*5)*3" se crea una tabla con una fila de encabezado (th) de 5 columnas, y 3 filas de celdas normales (td):

Tabla de encabezado de columna

De igual manera que se puede establecer una fila de encabezados en la tabla, también se puede crea una columna con encabezados, aunque su declaración es un poco diferente.
Por ejemplo, "table>(tr>th+(td*4))*4" creará una tabla con una columna de encabezados:

File Celda1 Celda2 Celda3 Celda4
Edit Celda1 Celda2 Celda3 Celda4
Selection Celda1 Celda2 Celda3 Celda4
View Celda1 Celda2 Celda3 Celda4

Tabla con combinaciones de celdas y título

Las celdas se pueden combinar, es decir, juntar entre sí, ya sea entre columnas ("colspan") o filas ("rowspan"). Incluso se pueden combinar.
Además, se puede añadir un título a la tabla con el comando "caption", el cual creará un título que se ajuste al ancho de la tabla:

Tabla customizada simple

Al igual que con el texto, las filas y columnas se puede editar. Para ello, se crean clases en el archivo .css, y al igual que con las listas, se puede afectar a filas y columnas concretas, siendo la fila de encabezados la más facil de editar.
En este caso va a ser algo simple como ejemplo.

Celda1 Celda2 Celda3 Celda4 Celda5

Como vemos, se pueden hacer muchas cosas con cada una de las celdas de la tabla. En la configuración más general de la tabla ("table.tcenteredsimple") se definen las características más generales de las celdas de la tabla, como los márgenes laterales, o el color del fondo.
Sin embargo, para cada celda que necesitase una configuración concreta, se pueden establecer características que sobreescribirán las generales dadas anteriormente.
Además, con colspan y rowspan se combinan celdas para evitar duplicados.
